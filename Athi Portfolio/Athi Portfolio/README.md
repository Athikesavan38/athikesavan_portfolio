# 🚀 Athikesavan's Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Full Stack Web Developer and Computer Science Engineering student.

## ✨ Features

- **Responsive Design** - Works perfectly on all devices
- **Dark/Light Mode** - Toggle between themes
- **Smooth Animations** - Professional scroll and hover effects
- **Mobile Menu** - Hamburger menu for mobile devices
- **Contact Form** - Integrated form with success animations
- **SEO Optimized** - Meta tags for better search visibility
- **Fast Loading** - Optimized performance with loading animation

## 📁 Files

- `index.html` - Main portfolio page
- `script.js` - JavaScript functionality
- `img.png` - Profile image
- `Resume.pdf` - Downloadable resume
- `favicon.svg` - Browser tab icon (SVG)
- `create-favicon.html` - Optional: Generate PNG favicon

## 🎨 Sections

1. **Hero** - Introduction with profile image
2. **About** - Personal information and contact details
3. **Education** - Academic background and achievements
4. **Skills** - Technical skills organized by category
5. **Projects** - Portfolio of work and projects
6. **Contact** - Get in touch form
